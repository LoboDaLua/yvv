# SoundFonts

Place a General MIDI `.sf2` SoundFont file in this directory for high-quality audio synthesis.

## Recommended free SoundFonts

- **FluidR3_GM** — https://member.keymusician.com/Member/FluidR3_GM/index.html
- **GeneralUser GS** — https://schristiancollins.com/generaluser.php
- **MuseScore_General** — bundled with MuseScore

Rename the file to `default.sf2` or the application will use the first `.sf2` found in this folder.

If no SoundFont is available, the application falls back to basic sine-wave synthesis.
